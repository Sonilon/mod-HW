# ChatGuard — Помощник модератора чата
### Fabric 1.20.1 | fabric-api 0.92.9+1.20.1

---

## 🔧 Как собрать мод

### Требования
- **Java 17+** (JDK, не JRE)
- **Gradle** — скачается автоматически через gradlew

### Шаги
```bash
# 1. Распакуй архив
cd chatguard-mod

# 2. Скачай Gradle Wrapper (только первый раз)
gradle wrapper --gradle-version 8.4

# 3. Собери мод
./gradlew build        # Linux/Mac
gradlew.bat build      # Windows
```

Готовый `.jar` появится в:
```
build/libs/chatguard-1.0.0.jar
```

Скопируй его в папку `.minecraft/mods/`

## 🎮 Функции мода

### Правый Shift → Открыть настройки
Нажми **[Right Shift]** в игре (не в чате) чтобы открыть панель настроек.

### 📋 Вкладки настроек

| Вкладка | Описание |
|---------|----------|
| ⚡ Управление триггерами | Список запрещённых слов с настройкой мута |
| 🔊 Звук | Включить/выключить звук оповещения |
| ✎ Формат команды мута | Шаблон `/tempmute {nick} {time} {reason}` |
| ✎ Формат сообщений | Как выглядит оповещение о нарушителе |
| ⚙ Паттерн ника | Regex для парсинга форматов чата сервера |

## ⚡ Триггеры

Каждый триггер содержит:
- **Слово** — что ищем в сообщениях (без учёта регистра)
- **Время** — длительность мута (30m, 1h, 2d...)
- **Правило** — пункт правил сервера (1.1, 2.3...)
- **Причина** — текст причины мута

**ЛКМ по триггеру** = редактировать  
**ПКМ по триггеру** = удалить

---

## 📨 Как работает обнаружение

1. Любое сообщение чата проверяется на триггеры
2. Если найдено запрещённое слово → мод извлекает ник из формата:
   ```
   [L/G | VIP] NickName [Титул]: сообщение
   ```
3. В чате появляется **красное кликабельное оповещение**:
   ```
   ════════════════════════════════════
   ⚠ НАРУШИТЕЛЬ: NickName — «слово» (Правило 1.1)
     [Нажмите чтобы открыть команду мута]
   ════════════════════════════════════
   ```
4. **Нажав на сообщение** — в строке чата появится команда:
   ```
   /tempmute NickName 1h Нецензурная брань (п.1.1)
   ```
5. Нажимаешь **Enter** — нарушитель замьючен!

```
┌─────────────────────────────┐

```
Пропадает через 6 секунд.

## 📁 Структура конфига

Конфиг хранится в `.minecraft/config/chatguard.json`:
```json
{
  "soundEnabled": true,
  "soundVolume": 1.0,
  "muteCommand": "/tempmute {nick} {time} {reason}",
  "violationFormat": "§c[ChatGuard] §f{nick} §7— §e{word} §7(Правило {rule})",
  "nickRegex": "(?:\\[[^\\]]*\\]\\s*)?([A-Za-z0-9_]{3,16})(?:\\s*\\[[^\\]]*\\])?:",
  "triggers": [
    {
      "word": "реклама",
      "time": "1h",
      "rule": "2.3",
      "reason": "Реклама сторонних ресурсов",
      "enabled": true
    }
  ]
}
```

## 🎨 Цветовые коды Minecraft (для форматов)
| Код | Цвет |
|-----|------|
| §c  | Красный |
| §e  | Жёлтый |
| §a  | Зелёный |
| §f  | Белый |
| §7  | Серый |
| §l  | Жирный |
| §r  | Сброс |

## ⚙ Кастомизация формата чата

Если на твоём сервере нестандартный формат чата, настрой **Regex паттерн ника** в настройках.

**Примеры:**
- `[L/G | VIP] NickName [Мастер]: текст`  → `(?:\[[^\]]*\]\s*)+([A-Za-z0-9_]{2,16})(?:\s*\[[^\]]*\])*\s*:`
- `NickName: текст` → `^([A-Za-z0-9_]{2,16})\s*:`

В настройках есть **встроенный тест** — вставь пример сообщения и проверь что ник распарсился правильно.
